# Dev-Muller-Portifolio

Personal portfolio site built with Next.js + Tailwind CSS. The project list
on the homepage is fetched live from the [GitHub API](https://api.github.com/users/Dev-Muller/repos)
on each page load/revalidation, so it always reflects current public repos.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
